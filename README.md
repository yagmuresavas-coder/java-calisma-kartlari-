Interactive Developer FlashcardsCodeCards, JAVA yazılım dillerindeki temel kavramları,
anahtar kelimeleri ve kod bloklarını görselleştirerek öğrenmeyi hızlandıran modern bir masaüstü uygulamasıdır.

✨ Özellikler

Modern UI: Karanlık mod (Dark Mode) odaklı,göz yormayan şık arayüz.
Kategorizasyon: Keyword, Modifier, Class gibi yapıların renk kodlarıyla ayrıştırılması.
Etkileşimli Kartlar: Tıklandığında dönen (Flip) ve detaylı kod örneğini gösteren kart yapısı.
Anlık Filtreleme: Arama çubuğu sayesinde istenilen kavrama saniyeler içinde ulaşım.
Dinamik Veri: Verilerin dış bir dosyadan (JSON) yönetilebilir olması.

🛠️ Kullanılan Teknolojiler 

Dil: Java 17+Arayüz (UI)JavaFXStilCSS (Custom Dark Theme)
Bağımlılık Yönetimi: MavenIDEIntelliJ IDEA

📂 Proje YapısıPlaintextsrc/main/java/com/codecards/app/
├── model/       # Veri modelleri (Card sınıfı vb.)
├── controller/  # UI etkileşimlerini yöneten mantık
└── ui/          # Görünüm katmanı ve FXML dosyaları

src/main/resources/
├── style/       # Uygulama CSS dosyaları
└── data/        # Kart bilgilerini tutan JSON verisi
